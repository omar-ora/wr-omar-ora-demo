This tool implements the Monte Carlo Tree Search (MCTS) algorithm to identify the optimal assignment sequence of company-owned equipment across eight active projects. With 8 projects, there are 40,320 possible sequences (8! = 40,320) — a space too large for intuitive evaluation or exhaustive enumeration in real time. The MCTS algorithm explores this space in 5,000 directed iterations using the UCT (Upper Confidence Bound for Trees) selection criterion, concentrating computational effort on the most promising branches of the decision tree. All cost fields ($) are in US dollars. Time fields in days from today (Day 0). Distance fields in miles. All outputs are computed by the algorithm — nothing is hardcoded.
